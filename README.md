# Social-Media-Addiction-DataMining
### Social Media Addiction Analysis Using Data Mining Techniques

## Project Description
This project aims to analyze social media addiction patterns using data mining techniques.Classification (Decision Trees) is used to predict user addiction levels (High, Medium, Low), while Clustering (K-means) is applied to identify groups of users with similar digital behaviors and engagement characteristics.

## Motivation
Social media addiction is a growing global concern affecting mental health and productivity.Understanding the behavioral factors that lead to excessive usage helps in developing digital well-being strategies and predictive models for early intervention.
